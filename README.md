## HMS Setup Instructions

### Compiling and Running the Project

#### Through the Terminal

Follow these instructions to clone, compile, and run the project:

1. **Clone the Repository:**
   ```
   git clone https://github.com/nithinsankar-b/SC2002_HMS_Team1.git
   ```

2. **Navigate to the Repository:**
   ```
   cd "SC2002_HMS_Team1"
   ```

3. **Compile the Project:**
   ```
   javac -cp src -d bin src/Main.java
   ```

4. **Run the Project:**
   ```
   java -cp bin Main.main
   ```

```bash

PASSWORD: patient11

# Patient 2
PATIENTID: P1012
PASSWORD: patient12

PATIENTID: P1013
PASSWORD: patient13
```

**Doctors:**
```bash

DOCTORID: D001
PASSWORD: doctor1

# Doctor 2
DOCTORID: D002
PASSWORD: doctor2
```

**Pharmacists:**
```bash
# Pharmacist 1
PHARMACISTID: PH001
PASSWORD: pharmacist1
```

**Administrator:**
```bash
# Admininistrator (Only 1)
ADMINID: A001
PASSWORD: administrator1
```
